`commands/career.md` 와 동일하게 동작해.

## 사용법

```
/취업                       # 잡 헌팅 (default)
/취업 자소서 마키나락스      # 자소서 작성
/취업 면접 마키나락스        # 면접 대비
/취업 풀준비 마키나락스      # ⭐ 풀 파이프라인 (4단계 자동)
/취업 진척                  # 취업 활동 추적
/취업 빠르게                # 빠른 모드 (CEO 압축)
```

## 핵심 모드: 풀준비 ⭐

`/취업 풀준비 [회사]` 한 명령어로:

1. 회사 풀 분석 (병렬 검색)
2. 자소서 작성 (Bar-Raiser 자체 검증)
3. 면접 대비 (Bar-Raiser 자체 검증)
4. 통합 패키지 출력
5. 메타 Bar-Raiser (자소서 ↔ 면접 일관성 검증)

→ 단발성 명령 3개를 1개로. 30분 → 5분.

자세한 워크플로우는 `commands/career.md` 참조.
