## 워크플로우 요약

처음 student-kit 사용하는 사람을 위한 8단계 질문:

1. 이름
2. 소속 (학교/학년/전공)
3. **진로 목표** ⚠️ CEO 모드 - 추상적 답변 거부
4. 응답 스타일 (반말/존댓말, 직설/친절)
5. 메인 작업 폴더 (스킵 가능)
6. GitHub username (스킵 가능)
7. 절대 금지 사항 (스킵 가능)
8. 현재 마감 일정 (스킵 가능)

→ 답변 요약 확인 → CLAUDE.md 자동 생성 → 다음 단계 안내
